Horizons App

Horizons is an iOS application that allows anyone to explore the world around them through user-sourced images. Users can comment on, like, save, and share other user's shared images, as well as filter images and follow other users. Be sure to check it out in the app store!

DodgeTheClouds

DodgeTheClouds is an iOS application that incorporates custom Bezier Paths, Core Graphics animations, and gesture recognizers to allow the user to maneuver an airplane to avoid oncoming clouds.

PGNColors

PGNColors is a CocoaPod that provides 40+ baked-in UIColors as well as a convenience method to convert hex colors to UIColors and a method that simplifies color gradients.

Spit. For all of your freestyle rapping needs.

Spit uses AVAudioPlayer to provide a beat and AVSpeechSynthesizer to read generated words so that the user can practice freestyle rapping. This was a group project with Haaris Muneer, presented at a Meetup at the Flatiron School.
